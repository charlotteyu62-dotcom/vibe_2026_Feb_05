
# Travel Advisor Chatbot

## Overview

A web-based chatbot that provides travel advice for countries in the Asia-Pacific region.

## Features

*   Interactive chat interface.
*   Provides information on:
    *   Travel Restrictions
    *   Dos and Don'ts
    *   Dress Code
    *   Alcohol Consumption
    *   LGBT Safety
    *   Food Recommendations
    *   Hotel Recommendations

## Current Task

Convert the existing "Toto Number Generator" application into a Travel Advisor Chatbot.

### Plan

1.  **Modify `index.html`**: Update the HTML structure to replace the `<toto-generator>` with a `<chat-bot>` custom element.
2.  **Modify `main.js`**: Create a `<chat-bot>` web component to manage the chat interface and responses.
